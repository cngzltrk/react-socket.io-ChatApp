# Chat App React - Nodejs - Socket.io

Simple Chat app

Project run 

	-npm install //in mychatapp for react 
	-npm install // in server for node js
	
	-npm start // for react 
	-npm start //for node.js
 
## Front END
  - React 
  - CSS
  - Reactstrap
  
## Back end
  - Node.js

#### Library

	- Socket.io
	- Reactstrap
	- Nodemon //for nodejs rerunning when file change
